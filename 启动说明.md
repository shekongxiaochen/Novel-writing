# 启动说明

以后用户说“启动前后端”，默认按这里执行，不再重新摸索。

## 后端

- 工作目录：`D:\Novel writing\backend`
- 启动命令：

```powershell
D:\Novel writing\backend\.venv\Scripts\python.exe -m uvicorn app.main:app --host 127.0.0.1 --port 8000
```

- 健康检查：`http://127.0.0.1:8000/health`
- 环境文件：`D:\Novel writing\backend\.env`

## 前端

- 工作目录：`D:\Novel writing\frontend`
- 启动命令：

```powershell
npm.cmd run dev -- --host 127.0.0.1
```

- 默认地址：`http://127.0.0.1:5174`

## 这台机器上更稳的后台启动方式

如果需要把前端挂到后台，优先直接用 Node 启动 Vite，不走 `npm.cmd` 的后台转发：

```text
E:\Download\nodejs\node.exe D:\Novel writing\frontend\node_modules\vite\bin\vite.js --port 5174 --host 127.0.0.1
```

## 当前约定

- 前端端口：`5174`
- 后端端口：`8000`
- 后端起来后，用 `/health` 探活。
- 前端起来后，访问首页确认返回 `200`。
- 日志文件：
  - `D:\Novel writing\backend\uvicorn.log`
  - `D:\Novel writing\backend\uvicorn.err.log`
  - `D:\Novel writing\frontend\vite.log`
  - `D:\Novel writing\frontend\vite.err.log`
