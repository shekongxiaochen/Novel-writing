# 启动说明

后端仅保留 **Rust**（目录 `backend/`），默认端口 **8080**。需本机 **MySQL**、**Redis**，Rust 工具链见 `D:\novel-dev\README.md`。

## 后端（Rust）

- 工作目录：`D:\Novel writing\backend`
- 环境文件：`D:\Novel writing\backend\.env`
- 启动（需先启动 Redis）：

```powershell
$env:RUSTUP_HOME = "D:\novel-dev\rustup"
$env:CARGO_HOME = "D:\novel-dev\cargo"
$env:Path = "D:\novel-dev\cargo\bin;" + $env:Path
cd "D:\Novel writing\backend"
cargo run
```

或：`D:\novel-dev\start-rust-backend.ps1`（脚本内路径需指向 `backend`）

- 健康检查：`http://127.0.0.1:8080/health`
- 数据库检查：`http://127.0.0.1:8080/health/db`
- **管理后台**（axum-admin）：`http://127.0.0.1:8080/admin/`（详见 `docs/admin-console.md`）

在 `backend/.env` 中启用：

```env
ADMIN_ENABLED=true
ADMIN_USERNAME=admin
ADMIN_PASSWORD=请改为强密码
```

默认账号为 `admin` / `change-me-admin`（未设置 `ADMIN_PASSWORD` 时）。

## 前端

- 工作目录：`D:\Novel writing\frontend`
- API 地址：`frontend\.env.local` 中 `VITE_API_BASE_URL=http://127.0.0.1:8080`

```powershell
cd "D:\Novel writing\frontend"
npm run dev -- --host 127.0.0.1
```

- 默认地址：`http://127.0.0.1:5174`

## Redis（D 盘便携版）

```powershell
D:\novel-dev\start-redis.ps1
```
