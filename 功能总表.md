# 小说结构理清助手 — 功能总表

> 用途：产品范围与模块一览（不含实现方式）。可按「第一期 / 第二期 / 第三期」裁剪。

---

## 一、作品与写作

| 功能模块 | 说明 |
|----------|------|
| 新建作品 | 创建小说项目，进入后续配置与写作 |
| 作品元数据 | 书名、简介、题材/标签、叙事视角与基调、多线与否等约定、作者备忘 |
| 章 / 卷结构 | 卷、章、节的组织、排序、重命名 |
| 正文写作 | 日常码字主界面（对标常见写小说软件的基础能力） |
| 章级笔记 / 备忘 | 本章写作提示、待改点、与大纲的对应说明 |
| 检索与导航 | 按标题或关键词跳转章节、侧边目录 |
| 字数与进度 | 字数统计、可选的篇幅或阶段目标（概念级） |
| 导出 / 备份 | 正文与设定导出、备份（具体格式后续定） |

---

## 二、脉络与结构

| 功能模块 | 说明 |
|----------|------|
| 大纲 | 全书 / 卷 / 章级情节要点与层级 |
| 故事线 / 时间线 | 多线叙事、先后顺序、因果与并行关系 |
| 冲突与目标 | 主线矛盾、阶段目标、关键转折点（可与大纲联动） |
| 节奏与篇幅（可选） | 段落快慢、预计字数等（偏进阶） |
| 思维导图（预留） | 以导图形式整理情节或主题分支（实现阶段再定形态） |
| 故事线可视化（预留） | 与故事线数据同源的不同视图 |

---

## 三、地理与世界观

| 功能模块 | 说明 |
|----------|------|
| 世界观总览 | 时代背景、规则体系、社会结构等总述 |
| 设定库 | 可检索的设定条目：地点、物品、规则、名词表等 |
| 地图（预留） | 地理关系、路线、势力分布等（功能占位，实现后续展开） |
| 设定关联 | 设定与章节 / 大纲 / 情节的引用关系（概念上减少吃书） |

---

## 四、角色与势力

| 功能模块 | 说明 |
|----------|------|
| 角色档案 | 姓名、性别、年龄、外貌、性格、目标、秘密、成长弧等 |
| 角色关系网 | 人物之间的关系类型及变化（亲友 / 敌对 / 利用等） |
| 势力 / 组织档案 | 名称、目标、资源、领袖、与主角关系 |
| 势力关系网 | 联盟、敌对、从属、隐藏关系等 |
| 出场与线索（可选） | 角色在哪些章 / 情节出现，便于查遗漏 |

---

## 五、质量、伏笔与待办

| 功能模块 | 说明 |
|----------|------|
| 伏笔与回收 | 埋设点、计划回收位置、是否已回收 |
| 坑点 / 疑问清单 | 逻辑漏洞、未解释点、时间线疑问、人设是否 OOC 等 |
| 矛盾清单 | 已发现的设定冲突、待决策点 |
| 写作待办 | 本章任务、待补设定、待改前文 |

---

## 六、进阶「功课型」（可选模块）

| 功能模块 | 说明 |
|----------|------|
| 主题与母题 | 全书想表达的核心，避免情节发散 |
| 核心梗概 | 开端—发展—高潮—结局梗概，用于对照大纲 |
| 人物弧光表 | 主要角色从 A 到 B 的状态与关键转折 |
| 场景清单 | 重要场景的目的（推进情节 / 揭示人物 / 铺垫） |
| 资料与参考 | 外部链接、摘抄、灵感库，与正文分离存放 |

---

## 七、账号与协作（后续）

| 功能模块 | 说明 |
|----------|------|
| 登录 / 账号 | 单用户先行，后续可扩展 |
| 多作者协作（后续） | 权限、分工、评论或审阅（范围待定） |

---

## 八、分阶段建议（仅作范围参考）

| 阶段 | 建议纳入范围（概念） |
|------|----------------------|
| 第一期 | 作品与元数据、章 / 卷与正文写作、大纲、角色与势力档案、伏笔 / 坑点 |
| 第二期 | 故事线 / 时间线、角色关系网与势力关系网（可先表格化再可视化）、设定库强化 |
| 第三期 | 思维导图、地图等可视化、AI 辅助、多作者协作 |
